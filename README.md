# Tools chic-chat
Express js  
MySQL  
React js  


# Installation  

## Database

Create Database name chic_chat  
```
> CREATE DATABASE IF NOT EXISTS chic_chat
```

Config server in config.js  

## Back-end
```
> cd chic-chat\server
> npm install  
> node create_table.js  
> node server.js  
```



## Front-end 
```
> cd chic-chat\client  
> npm install 
> npm start
```
