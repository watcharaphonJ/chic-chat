# Tools chic-chat
Express js  
MySQL  
React js  


# Installation  

## Database

Create Database name chic_chat  
'''
> CREATE DATABASE IF NOT EXISTS chic_chat
'''
and  
Config server in config.js  

## Back-end
```
> cd chic-chat\server
> npm install  
> npm install mysql  
> node create_db.js  
> node db_connection.js  
```



## Front-end 
```
> cd chic-chat\client  
> npm install 
> npm start
```
